# 交通費 電卓 — 公開手順

ITに詳しくない人がスマホで使える、交通費の集計アプリです。
運賃を入れて足すと1日の合計が出て、確定すると月のリストが自動でたまります。
データは各自のスマホの中だけに保存され、アカウントは不要です。

## 含まれるファイル
- `index.html` … アプリ本体
- `manifest.webmanifest` … ホーム画面追加用の設定
- `icon-180.png` / `icon-192.png` / `icon-512.png` … アプリのアイコン

この4種（5ファイル）を、すべて同じ場所（リポジトリの直下）に置いてください。

## GitHub Pages で公開する（作る人が一度だけ）
1. GitHub で新しいリポジトリを作成（例: `transit-calc`）。Public でOK。
2. 上記のファイルをすべてリポジトリ直下にアップロード（ドラッグ＆ドロップで可）。
3. リポジトリの Settings → Pages を開く。
4. Build and deployment の Source を「Deploy from a branch」にし、Branch を `main`／フォルダを `/ (root)` にして Save。
5. 1〜2分待つと、ページ上部に公開URLが表示される。
   形式: `https://<ユーザー名>.github.io/<リポジトリ名>/`
6. そのURLをスマホで開いて動作確認する。

## iPhone でホーム画面に追加する（使う人）
1. Safari で公開URLを開く。
2. 画面下の「共有」ボタン（□に↑）をタップ。
3. 「ホーム画面に追加」を選ぶ。
4. 追加すると、ホーム画面にアプリのアイコンができる。次回からはアイコンを押すだけ。

Android（Chrome）の場合は、メニュー（︙）→「ホーム画面に追加」で同様に追加できます。

## 使い方
- 運賃を入力して「＋ 足す」。乗った分だけ足すと、上にその日の合計が出ます。
- 日付は初期で今日。Suicaの反映が遅れたときは、日付を選んで後から入れてもOK。
- 「この日を確定して保存」で、下の月リストに記録されます。
- 月末は「この月のリストを共有」で、メモ・メール・LINEなどへそのまま送れます。

## メモ
- 私用（買い物など）は入れない運用にすれば、集計は交通費だけになります。
- データはスマホの中（ブラウザの保存領域）にあります。アプリを消したり別の端末に変えると引き継がれません。
- 完全オフライン対応（電波がなくても起動）が必要なら、サービスワーカーを追加できます。
